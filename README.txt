# TebMed PWA Package

## Files
- manifest.json
- sw.js
- pwa_html_snippets.html

## How To Use

1. Copy manifest.json and sw.js into your project root.

2. Add the code from pwa_html_snippets.html into your HTML file.

3. Create an icons folder:

icons/
  icon-192.png
  icon-512.png
  maskable-512.png

4. Run project on:
- localhost
or
- HTTPS

5. Test in Chrome:
DevTools → Application → Manifest
